# MNISTラベルノイズ影響調査プロジェクト

本プロジェクトは、MNISTデータセットを用いた二値分類タスクにおいて、学習データに含まれるラベルノイズ（誤りデータ）の量と種類が、ニューラルネットワークの性能にどのような影響を与えるかを体系的に調査・分析することを目的とします。

## 1. 前提条件

*   Python 3.8 以上がインストールされていること
*   `pip` が利用可能であること

## 2. セットアップ手順

以下の手順に従って、プロジェクトの実行に必要な仮想環境を構築し、ライブラリをインストールします。

### ステップ1: 仮想環境の作成

プロジェクトのルートディレクトリで以下のコマンドを実行し、`venv`という名前の仮想環境を作成します。

```bash
python3 -m venv venv
```

#ステップ2: 仮想環境のアクティベート
作成した仮想環境を有効化します。OSによってコマンドが異なります。
macOS / Linux の場合:
code
```Bash
source venv/bin/activate
```
Windows (コマンドプロンプト) の場合:
code
```Bash
.\venv\Scripts\activate
```
アクティベートに成功すると、ターミナルのプロンプトの先頭に (venv) と表示されます。
ステップ3: 依存ライブラリのインストール
requirements.txtに記載されたライブラリをインストールします。後述のシェルスクリプトを実行するか、以下のコマンドを手動で実行してください。
code
```Bash
pip install -r requirements.txt
```

これでセットアップは完了です。

## 3. 実験の実行方法

プロジェクトのルートディレクトリで、以下のコマンドを実行します。

```bash
python main.py
```

実行が開始されると、config/experiment_config.yaml の設定に基づいた実験が自動的に進行し、進捗バーが表示されます。
4. 設定の変更
実験の範囲（対象ペア、エラー率など）やモデルのパラメータは、config/experiment_config.yaml ファイルを編集することで、コードに触れることなく自由に変更できます。
例：
全ペアで実験する場合: target_pairsをnullに変更します。
学習エポック数を変更する場合: training_params内のepochsの値を変更します。
5. 結果の確認
実験が完了すると、すべての結果は config.yaml で指定された results_file（デフォルトでは results/experiment_outputs.yaml）にYAML形式で出力されます。
